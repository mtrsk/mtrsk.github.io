Quorum September 25, 2024 by Marcos Benevides

:ID: 42950824-228d-48d6-abec-4e18908dfca0

Quorum means “majority”, and the idea behind quorum is easy to understand: if the Clusters doesn’t have quorum, no actions will be taken in the cluster. This by itself would offer a good solution to avoid the Split Brain problem.

But to make sure that it can never happen, that multiple nodes activate the same resources in the cluster, another mechanism is used as well. This mechanism is known as Stonith. (, Van Vugt 2014, 6)

This is required to avoid corruption on vital Clusters resources, such as the File System.

References

Van Vugt, Sander. 2014. Pro Linux High Availability Clustering. Apress.
